# Zasady Cytowania Zrodel Prawa Zywnosciowego PL/EU

## 1. Rozporzadzenia i dyrektywy UE

### Format pelny (pierwsze przywolanie)
```
Rozporzadzenie Parlamentu Europejskiego i Rady (UE) nr 1169/2011 z dnia 25 pazdziernika 2011 r.
w sprawie przekazywania konsumentom informacji na temat zywnosci (Dz.Urz. UE L 304 z 22.11.2011, s. 18)
```

### Format skrocony (kolejne przywolania)
```
Rozp. (UE) nr 1169/2011
Rozp. 1169/2011
```

### Konkretny przepis
```
Art. 9 ust. 1 lit. b) Rozp. (UE) nr 1169/2011
Art. 14 ust. 1 Rozp. (WE) nr 178/2002
Zalacznik II do Rozp. (UE) nr 1169/2011
```

### Wersja skonsolidowana
```
Rozp. (UE) nr 1169/2011 w brzmieniu skonsolidowanym z 14.05.2024
(CELEX: 02011R1169-20240514)
```

### Numer CELEX
```
CELEX: 32011R1169 (akt bazowy)
CELEX: 02011R1169-20240514 (wersja skonsolidowana)
```

## 2. Polskie ustawy i rozporzadzenia

### Ustawa - format pelny (pierwsze przywolanie)
```
Ustawa z dnia 25 sierpnia 2006 r. o bezpieczenstwie zywnosci i zywienia
(t.j. Dz.U. z 2023 r. poz. 1448)
```

### Ustawa - format skrocony
```
u.b.z.z.
ustawa o bezpieczenstwie zywnosci
```

### Konkretny przepis ustawy
```
Art. 29 ust. 1 u.b.z.z. (Dz.U. z 2023 r. poz. 1448)
Art. 31 ustawy o bezpieczenstwie zywnosci i zywienia
```

### Rozporzadzenie wykonawcze
```
Rozporzadzenie Ministra Zdrowia z dnia [data] w sprawie [tytul] (Dz.U. [rok] poz. [numer])
```

## 3. Orzecznictwo TSUE (Trybunal Sprawiedliwosci UE)

### Wyroki TSUE
```
Wyrok TSUE z 12.06.2014 w sprawie C-156/13, Digibet i Albers, ECLI:EU:C:2014:1756, pkt 42
Wyrok TS z 15.11.2018 w sprawie C-667/17, Bauer, ECLI:EU:C:2018:911, pkt 58-62
```

### Opinie Rzecznika Generalnego
```
Opinia Rzecznika Generalnego [nazwisko] z [data] w sprawie C-[numer], ECLI:EU:C:[rok]:[numer], pkt [nr]
```

### Format ECLI
```
ECLI:EU:C:[rok]:[numer] (Trybunal Sprawiedliwosci)
ECLI:EU:T:[rok]:[numer] (Sad)
```

## 5. Orzecznictwo WSA

### Wyroki WSA
```
Wyrok WSA w Warszawie z 12.05.2023, sygn. VII SA/Wa 1234/22
Wyrok WSA w Gliwicach z 08.09.2022, sygn. III SA/Gl 567/22
```

- SA/Gl - Gliwice

- SA/Gd - Gdansk

## 7. Opinie EFSA

### Format
```
EFSA NDA Panel, 2011. Scientific Opinion on [tytul]. EFSA Journal 2011;9(4):2078, doi:10.2903/j.efsa.2011.2078
EFSA CONTAM Panel, 2023. [tytul]. EFSA Journal 2023;21(10):8340
```

### Skrot
```
Opinia EFSA, EFSA Journal [rok];[vol]([issue]):[numer]
```

## 8. Doktryna i komentarze

### Artykuly naukowe
```
[Autor], [Tytul artykulu], [Czasopismo] [rok], nr [numer], s. [strony]
Krawczyk T., Notyfikacja suplementow diety - problemy praktyczne, Prawo i Medycyna 2020, nr 3, s. 45-67
```

### Komentarze
```
[Autor], w: [Redaktor] (red.), [Tytul komentarza], [Wydawnictwo] [rok], art. [nr], nb. [nr]
```

### Monografie
```
[Autor], [Tytul], [Wydawnictwo], [Miejsce] [rok], s. [strony]
```

## 9. Wytyczne i dokumenty niewiazace

### Wytyczne Komisji Europejskiej
```
Komisja Europejska, Zawiadomienie w sprawie [temat] (Dz.Urz. UE C [numer] z [data], s. [nr])
Commission Notice on [temat], 2017/C 428/01
```

### Dokumenty robocze
```
Dokument roboczy sluzb Komisji, [tytul], [sygnatura], [data]
```

### Wytyczne EFSA
```
EFSA, Guidance on [temat], EFSA Journal [rok];[numer]
```

## 10. Zasady ogolne

### Hierarchia cytowania
1. Przepis UE (rozporzadzenie > dyrektywa > decyzja)
2. Przepis krajowy (ustawa > rozporzadzenie)
3. Orzecznictwo (TSUE > NSA > WSA)
4. Opinie EFSA
5. Wytyczne / soft law
6. Doktryna

### Oznaczenia
- t.j. = tekst jednolity
- zm. = zmieniony
- uchylony = juz nie obowiazuje
- art./Art. = artykul
- ust. = ustep
- pkt = punkt
- lit. = litera
- zd. = zdanie
- par. = paragraf (w KPA)
- nb. = numer brzegowy (komentarze)
- pkt = punkt (w orzeczeniach TSUE)
- Rn. = Randnummer (orzeczenia niemieckie)

### Skroty aktow
- u.b.z.z. = ustawa o bezpieczenstwie zywnosci i zywienia
- u.PIS = ustawa o Panstwowej Inspekcji Sanitarnej
- KPA = Kodeks postepowania administracyjnego
- PPSA = Prawo o postepowaniu przed sadami administracyjnymi
- Rozp. = Rozporzadzenie
